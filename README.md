# 한국교육경영(KEM) 석박사 2026-2 커리큘럼 대시보드

aSSIST 한국교육경영학 석박사 과정의 2026년 2학기 커리큘럼 관리를 위한 인터랙티브 대시보드입니다.

## 🌐 바로 보기

GitHub Pages 배포 후 아래 URL로 접속:

```
https://<본인-GitHub-아이디>.github.io/<레포-이름>/
```

## 📋 주요 기능

- **월별 통합 달력**: 2026년 9월~12월, 주간·야간·토요일 합반을 색상 구분하여 한눈에 표시
- **26-2 신입생 시간표**: 석사·박사 × 주간·야간 4가지 조합의 표준 1학기 시간표
- **학생별 대시보드**: 재학·신입학 25명의 이수학점, 한국어교원자격증 영역별 진행도, 추천 수강 과목
- **상세 시간표**: 주간·야간 개설 과목별 일자·시간 상세 뷰
- **박사 연구방법론 토요 합반**: 주간+야간 통합 박사 3학기 이상 대상
- **전체 현황 요약**: 학생 전원의 이수 현황 테이블

## 📂 파일 구조

```
.
├── index.html      # 대시보드 본체 (데이터 내장, 단일 파일)
└── README.md       # 이 파일
```

모든 데이터가 `index.html` 하나에 포함되어 있어 추가 서버·API가 필요 없습니다.

## 🎓 포함된 학생 범위 (25명)

- 26-1 신입 석사 12명 (석사 8기)
- 석사1.5 과정 3명
- 박사 10명 (박사 1기~7기, 2025-2 이전 입학 포함)

2025-2 이전 입학 석사(석사 6기·7기)는 이미 26-1 졸업 예정이라 제외되었습니다.

## 🔄 데이터 업데이트

학생 데이터·개설 과목을 수정하려면 `index.html` 상단의 `const APP_DATA = {...}` JSON 객체를 편집하거나, 원본 Excel에서 다시 생성한 HTML로 파일을 교체하면 됩니다.
